I want to create a Dashboard for US institutions comparison. The audience will be institutions. It is called 'P2P - Pears to Peers: Benchmarking among equals'. An pears with pears comparison ensures that performance, quality, or outcomes are evaluated among true peers — those operating under similar conditions, scales, and contexts.

These are the data in the 'Data' folder:
- institutions_info_df_selected: columns ['institution_id', 'institution_name', 'country_code', 'type', 'ror', 'grid', 'continent_name', 'country_name', 'latitude', 'longitude', 'research_level']
- institutions_relationships_df_selected: columns ['instituion_id','child']. One institutions is the aggregation of its child.
- df_topics: columns ['topic_id_scaled', 'topic_id', 'topic_name', 'subfield_id', 'subfield_name', 'field_id', 'field_name', 'domain_id', 'domain_name', 'keywords', 'summary', 'wikipedia_url', 'subfield_id_scaled', 'field_id_scaled', 'domain_id_scaled']. Each topic belongs to one subfield, which belong to one field, which belong to one domain.
- dict_institution_id_name: dictionary keys instituion_id and values institution name
Inside 'Data' folder there are 3 subfolders, corresponding to different granularity, called 'All' (data for each institution for the whole dataset), 'Domain' (data divided by domain),'Field' (data divided by fields). Inside each of them there are these files:
- RCA_df_selected: columns ['period_id', 'institution_id', 'field_id_scaled', 'domain_id', 'portfolio_composition', 'field_name', 'domain_name'], where for each time period (4 periods: {0: (1990, 1999), 1: (2000, 2009), 2: (2010, 2019), 3: (2020, 2024)}), each institution, each field, I have the corresponding RCA normalized value, that we call portfolio composition. For the 'All' and 'Domain' level, we have a value for each field, instead for the 'Field' level , we have a value for each subfield.
- institutions_stats_selected: columns ['year', 'institution_id', 'n_works', 'n_works_weighted', 'n_authors']. For each year, each institution, I have these info.
- institutions_stats_periods_selected: columns ['period_id', 'institution_id', 'n_authors', 'n_works']. Where for each year and each institution I have number of works and authors.
- edges_df_selected_dict_all: dictionary 4 keys ['avg_weig', 'origin', 'geom', 'geom2']. I want to use the 'geom2' entry. For each of them correspond a dataframe with columns ['similarity_score', 'institution_id', 'institution_id_neighbor','institution_id_scaled2', 'institution_id_scaled2_neighbor']. Where, for each institution, I have its neighbours (institutions most similar according to metric in the key) with the corrispondent similarity. Institution_id correspond to the instituion id in OpenAlex, instead institution_id_scaled is the id in the subset of institutions in US considered.
- entropy_df_selected: columns ['institution_id', 'period_id', 'disorder_index']. For each institution and period, we show the entropy of the corresponding portfolio. We call it disorder index.
- institution_peers_dict: dictionary key institution_id_scaled and values the corresponding neighbours.
- impact_selected: columns ['institution_id', 'year', 'relative_impact', 'impact_uniformity']. For each institution, and each year, it is regarding the impact.
Inside 'Data' folder there is also the subfolder 'Topic', which contain:
- RCA_topic_df_selected: ['topic_id_scaled', 'count_specialized_topic', 'institution_id', 'topic_id', 'domain_id_scaled', 'field_id_scaled', 'value']. Where we have the RCA at the topic level. 
- topics_embedding_reduced_df: columns ['topic_id_scaled', 'topic_id', 'feature_0', 'feature_1']. This is an embedding in 2 dimentions of the topics.
